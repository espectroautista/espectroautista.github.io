▲ Enlaces permanentes a resultados de tests

Jueves 2 de agosto de 2012

Se ha incorporado una nueva prestación a EspectroAutista.Info que será de gran ayuda a quienes deseen conservar los resultados obtenidos al cumplimentar cualquier test. Por ejemplo, los profesionales de la salud pueden aplicar tests a múltiples personas, incluso repitiendo alguno de ellos de vez en cuando, y les puede ser de gran utilidad poder disponer de una especie de base de datos con esos tests, presentados tal como si se acabasen de realizar.

Cuando se completa cualquier test y se visualiza el resultado obtenido, aparece junto a este un enlace permanente que puede ser conservado para recuperar el test en el futuro. Para ello el enlace se debe almacenar, ya sea en el propio navegador o copiando la dirección y enganchándola en un documento de texto, hoja de cálculo, base de datos, etc.

Estos son ejemplos de enlaces a tests con diferentes formas de presentación:

  • Test sin descripción.
  • Test con descripción.
  • Test en formato de impresión.

La dirección de estos enlaces contiene, tras el carácter #, diversos parámetros que los visitantes pueden en algunos casos modificar para obtener diferentes formas de presentación. Por ejemplo, en la cadena #imprimir:si;descripcion:no; se puede entender que las palabras si y no definen diversos comportamientos de la página, y que intercambiando esas palabras se obtendrá (o no) una página en formato de impresión, con o sin descripción, etc.

Aviso importante

La edición y mantenimiento de EspectroAutista.Info se ha descontinuado. Para conocer todos los detalles lea la última noticia.
